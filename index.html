Bootstrap link's to CDN

just copy links given below to your html code & enjoy the beautiful power of Bootstrap

CSS

Complete link for bootstrap css files:

 https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css

Ready made link tag for ready to use:

link rel="stylesheet" type="text/css" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css"

just copy whole line and paste it inside triangular brackets "<..code..>" in your head section of HTML page.


JavaScript

Complete link for bootstrap JavaScript files:

https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/js/bootstrap.min.js

Readymade scripts tag for ready to use:

src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/js/bootstrap.min.js" integrity="sha384-Tc5IQib027qvyjSMfHjOMaLkfuWVxZxUPnCJA7l2mCWNIpG9mGCD8wGNIcPD7Txa" crossorigin="anonymous

just copy whole line and insert it as attributes for script tag in head section of your html page.


Don't Forget To Add JQuery!!

Link for jquery plu-ins: 

src="https://code.jquery.com/jquery.js"

just copy the line given above and place it as attribute in your head section to a blank script tag other than your working one.
